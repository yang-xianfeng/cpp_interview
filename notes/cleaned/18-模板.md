# 模板
### 函数模板
#### 形式：
```cpp
template <typename T>
// <>括号中的参数叫模板形参，模板形参和函数形参很相像，模板形参不能为空。
T add(T x, T y) {
    cout << "T add(T, T)" << endl;
    return x + y;
}
```

注意：`class`与`typename`在此**没有任何区别，完全一致**。

`class`出现的时间比较早，通用性更好一些，`typename`是后来才添加了，对于早期的编译器可能识别不了`typename`关键字。

```cpp
#include <iostream>
using namespace std;

template <class T = int>
class A {
public:
    virtual void foo(T t) { cout << "A::foo()" << endl; }
};

template <class T = int, class R = char>
class B : public A<T> {
public:
    void foo(T t) override { cout << "B::foo()" << endl; }

private:
    R *r;
};

int main() {
    A<int> a;
    a.foo(0);// A::foo()

    B<int, char> b;
    A<int> *aPtr = &b;
    aPtr->foo(10);// B::foo()
}
```

#### 实例化  （ 函数模板 ---实例化---> 模板函数 ）

- 显式实例化
- 隐式实例化

```cpp
cout << "add(ia, ib) = " << add(ia, ib) << endl;//隐式实例化，没有明确说明类型，靠编译器推导
cout << "add(da, db) = " << add<double>(da, db) << endl;//显示实例化，编译器无序推导
```

普通函数可以与函数模板进行重载，且**普通函数优先级高**于函数模板。

函数模板也**可以重载**。

#### 注意：

对于模板而言，**声明和实现不能分开**，否则会报错。

如果一定要分开，可以在**头文件**中`**#include 实现文件**`。（ 与 `inline`类似 ）

模板的特化 `template <>`

模板的全特化：模板的参数列表 全部特化

```cpp
 template <int a, double b, string c>
```

模板的偏特化：模板的参数列表 至少有一个没有特化

```cpp
 template <int a, double b, typename T>
```

#### 函数模板的参数类型

```cpp
 // template <typename T, short kSize = 10>
template <short kSize = 10, typename T = int>
T multiply(T x, T y){
    return  x * y * kSize;
}

// 类型参数 typename
// 非类型参数，属于整型 short / int / long / long long / bool / void *  // 非类型参数 排除double / float两种浮点型。
非类型模板形参的形参和实参间所允许的转换:
1、允许从数组到指针，从函数到指针的转换。如：template <int *a> class A{}; int b[1]; A m;即数组到指针的转换
2、const修饰符的转换。如：template<const int *a> class A{}; int b; A<&b> m; 即从int *到const int *的转换。
3、提升转换。如：template class A{}; const short b=2; A m; 即从short到int 的提升转换
4、整值转换。如：template class A{}; A<3> m; 即从int 到unsigned int的转换。
5、常规转换。
```

#### 成员函数模板

```cpp
#include <iostream>

using std::cout;
using std::endl;

class Point {
public:
    template <typename T = double>
    Point(T x, T y) : _x(x), _y(y) {
        cout << "Point()" << endl;
    }
    //成员函数模板,成员函数模板也是可以设置默认值
    // template <typename T> // pt.func()会有问题
    template <typename T = int>
    T func() {
        return (T)_x;
    }

private:
    double _x;
    double _y;
};

int main(void) {
    Point pt(1.1, 2.2);
    cout << "pt.func() = " << pt.func<int>() << endl;
    cout << "pt.func() = " << pt.func<double>() << endl;
    cout << "pt.func() = " << pt.func() << endl;
    return 0;
}
```

### 类模板

```cpp
#include <iostream>
#include <string>

using std::cout;
using std::endl;
using std::string;

//类模板
template <typename T, size_t defaultSize = 10>
class Stack {
public:
    Stack() : _top(-1), _data(new T[defaultSize]()) {
        cout << "Stack()" << endl;
    }
    ~Stack();
    bool full() const;
    bool empty() const;
    void push(const T &value);
    void pop();
    T top() const;

private:
    int _top;
    T *_data;
};

template <typename T, size_t defaultSize>
Stack<T, defaultSize>::~Stack() {
    if (_data) {
        delete[] _data;
        _data = nullptr;
    }
}

template <typename T, size_t defaultSize>
bool Stack<T, defaultSize>::full() const {
    return _top == defaultSize - 1;
}

template <typename T, size_t defaultSize>
bool Stack<T, defaultSize>::empty() const {
    return _top == -1;
}

template <typename T, size_t defaultSize>
void Stack<T, defaultSize>::push(const T &value) {
    if (!full()) {
        _data[++_top] = value;
    } else {
        cout << "stack full" << endl;
        return;
    }
}

template <typename T, size_t defaultSize>
void Stack<T, defaultSize>::pop() {
    if (!empty()) {
        --_top;
    } else {
        cout << "stack empty" << endl;

        return;
    }
}

template <typename T, size_t defaultSize>
T Stack<T, defaultSize>::top() const {
    return _data[_top];
}

void test() { Stack<int, 20> st; }

void test2() { Stack<double, 10> st; }

void test3() { Stack<string> st; }

int main(int argc, char **argv) {
    test();
    return 0;
}
```


**函数**参数包必须唯一，且是**函数**的最后一个参数

打印实参列表

递归

- ...位于参数左边的时候，称为打包
- ...位于参数右边的时候，称为解包

```cpp
// 空参数列表 递归出口
void show(){
    cout << endl;
}

template <typename T, typename... Args>
void display(T t, Args... args) {
    cout << t << " ";
    display(args...);//当... 位于args右边的时候叫做解包
}
```
