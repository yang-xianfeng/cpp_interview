#### 1.6.2  fork 系统调用 
`fork` - create a child process<br />#include <sys/types.h><br />#include <unistd.h><br />pid_t fork(void);<br />fork()  creates a new process by **duplicating** the calling process.  <br />RETURN VALUE<br />       On  success, the PID of the child process is returned in the parent, and 0 is returned in the child.  
```c
#include <myhead.h>
int main(){
    pid_t pid ;
    pid = fork();
    if(pid == 0){
        printf("I am child, pid = %d, ppid = %d\n", getpid(), getppid());
    }
    else{
        printf("I am parent, pid = %d, ppid = %d\n", getpid(), getppid());
        sleep(1);
    }
}
```

`**fork**`**的拷贝**<br />在逻辑上，父子进程的用户态空间(** 栈 堆 数据段  FILE（文件流）**)是拷贝的 
```c
#include <myhead.h>
int global = 1;
int main()
{
    int stack = 2;
    int *pHeap = (int *)malloc(sizeof(int));
    *pHeap = 3;
    if (fork() == 0)
    {
        printf("I am child process, global = %d, stack = %d, *pHeap = %d\n",
               global, stack, *pHeap);
        ++global;
        ++stack;
        ++*pHeap;
        printf("I am child  process, global = %d, stack = %d, *pHeap = %d\n",
               global, stack, *pHeap);
    }
    else
    {
        sleep(5);
        printf("I am parent process, global = %d, stack = %d, *pHeap = %d\n",
               global, stack, *pHeap);
    }
}

/*  I am child  process, global = 1, stack = 2, *pHeap = 3
    I am child  process, global = 2, stack = 3, *pHeap = 4
    I am parent process, global = 1, stack = 2, *pHeap = 3 */
```

### 1.7 进程组
进程组：进程的集合（每个进程只能属于一个进程组， 组ID是组长的PID）

1. 父子进程 属于 同一进程组。
1. 组长终止，组ID不变
1. 新建进程不能与之前组里的iD重复
1. 普通组员可以创建新的组，但是**组长 不能**创建新的组
