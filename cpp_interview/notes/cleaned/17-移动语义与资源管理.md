## 移动语义
#### 左值与右值
左值：可以取地址
右值：不能取地址，包括临时变量、临时对象（string("word") ）、字面值常量。

> 临时对象会短暂的存在于**栈**上，生命周期极短。  —— 20年老程序员
> `&10 // 字面值常量，右值 Error`  短暂的存在于**栈**上

```c
#include<iostream>
int a = 1;//显示初始化全局变量，全局变量a的内存空间将会被分配在全局/静态变量区
char* p;//没有显示设置初始值，编译器自动为全局变量设置初始的零值，对于指针变量零值就是nullptr
const int cx = 10; // 文字常量区 (包括 字符串常量 和 const全局变量)
int main() {
     const int cy = 100; // 栈上
	int b;//非静态局部变量，空间分布在栈上，且初始值不定
	char s[] = "hello,world!";//非静态局部字符数组，数组元素分配在栈上
	char* p;//非静态局部指针变量，内存空间分配在栈上
	char* s1 = "c++ is best";//非静态局部指针变量s1内存空间在栈上，而字符串"c++ is best"存储在常量区
	static int c = 0;//局部静态变量，内存空间被分配在全局/静态变量区

	p = new char[10];//动态申请的10字节的内存空间被分配在堆上
	strcpy(p,"c++ is best";//字符串"c++ is best"存放在常量区，编译器可能会将它与s1所指向的的"c++ is best"优化成一个地方

	return 0;
}
```

左值引用： **左值**，**只能绑定左值**，不能绑定右值
const左值引用：既可以绑定左值，也可以绑定右值

`**&&**`**右值引用 ( C++11 )**：本身**可以是左值(**`**int &&rref = 10**`**) ，也可以是右值(函数的返回值是右值引用)**，不能绑定左值，**只能绑定到右值**（ 绑定以后本来会被销毁的右值的生存期会**延长到它的右值引用的生存期** ）

```cpp
void test{
    int a = 10;
    int b = 20;
    int *pflag = &a;
    vector<int> vec;
    vec.push_back(1);
    string s1 = "hello"; // 字符串常量，位于文字常量区，左值
    string s2 = "world"; // 字符串常量，位于文字常量区，左值

    // const 左值引用
    const int &ref = a;//const左值引用可以绑定到左值
    const int &ref1 = 10;//const左值引用可以绑定到右值
    &ref;
    &ref1;

    //右值引用
    const int &&rref = 100;//右值引用可以绑定到右值
    /* const int &&rref2 = b;//右值引用不能绑定到左值 */
    &rref;//此时右值引用是一个左值
    &a;//左值
    &b;
    &pflag;
    &(*pflag);
    &vec[0];
    &s1;
    &s2;
    /* &(a + b);//error,右值 */
    /* &(s1 + s2);//error,右值 */
    /* &(a++);//error; */
    &(++a);//ok
    /* &100;//error,右值，字面值常量 */
    /* &string("hello");//error,右值 */
}
```

#### 移动构造函数

少一次堆空间的申请操作和临时对象的释放操作

针对右值而言，移动构造函数的优先级 高于 拷贝构造函数的优先级

#### 移动赋值运算符函数

少一次堆空间的申请操作和临时对象的释放操作

`std::move` 可以将左值转换成右值。使用 `std::move` 需要考虑**自移动**的可能性。

注意：

-  拷贝构造函数与赋值运算符函数 **编译器会自动提供**。
-  移动构造函数与移动赋值运算符函数 编译器**不会提供**，必须自行提供。
-  拷贝构造函数与赋值运算符函数 称为具有拷贝控制语义的函数。
-  移动构造函数与移动赋值运算符函数 称为具有移动语义的函数。

**移动语义的函数** **优先级高**于 拷贝控制语义的函数。（**先尝试执行 移动语义的函数**，不行就是左值，再执行 拷贝控制语义的函数）

#### `std::move()`

将左值转换为右值，在内部其实上是做了一个强制转换， `static_cast<T& rvalue>(lvaule)`。将左值转换为右值后，左值就不能直接使用了，如果还想继续使用，必须重新赋值。

`std::move()`作用于内置类型没有任何作用，内置类型本身是左值还是右值，经过`std::move()`后不会改变。

### C++的解决办法( RAII 技术)

资源管理 RAII 技术，利用对象的生命周期管理程序资源(包括内存、文件句柄、锁等)的技术,因为对象在 离开作用域的时候，会自动调用析构函数。

关键：要保证资源的释放顺序与获取顺序**严格相反**。正好是析构函数与构造函数的作用。

RAII常见特征

1. 在构造时初始化资源，或者托管资源。
1. 析构时释放资源。
1. 提供若干访问资源的方法。
1. 一般不允许复制或者赋值(对象语义)    将拷贝构造函数和赋值运算符函数**设置为私有**或者**删除**

- 值语义：可以进行复制与赋值。
- 对象语义：不能进行复制与赋值。

```cpp
// raii.cc
#include <iostream>

using std::cout;
using std::endl;

class Point{
public:
    Point(int ix = 0, int iy = 0)
    : _ix(ix)
    , _iy(iy)    {
        cout << "Point(int = 0, int = 0)" << endl;
    }

    void print() const    {
        cout << "(" << _ix
             << ", " << _iy
             << ")";
    }

    ~Point()    {
        cout << "~Point()" << endl;
    }

    friend std::ostream &operator<<(std::ostream &os, const Point &rhs);

private:
    int _ix;
    int _iy;
};

std::ostream &operator<<(std::ostream &os, const Point &rhs) {
    os << "(" << rhs._ix
       << ", " << rhs._iy
       << ")";

    return os;
}

//模板相关的概念
template <typename T>
class RAII {
public:
    //在构造函数中初始化资源
    RAII(T *data)
    : _data(data)    {
        cout << "RAII(T *)" << endl;
    }

    //在析构函数中释放资源
    ~RAII()    {
        cout << "~RAII()" << endl;
        if(_data)  {
            delete _data;
            _data = nullptr;
        }
    }

    //提供若干访问资源的方法
    T *operator->() {
        return _data;
    }

    T &operator*() {
        return *_data;
    }

    T *get() {
        return _data;
    }

    void reset(T *data) {
        if(_data) {
            delete _data;
            _data = nullptr;
        }

        _data = data;
    }

    //不允许复制或者赋值
    RAII(const RAII &rhs) = delete;
    RAII &operator=(const RAII &rhs) = delete;
private:
    T *_data;
};

int main(int argc, char **argv) {

    RAII<int> raii(new int(10));//栈对象的生命周期

    cout << endl;
    RAII<Point> pt(new Point(1, 3));//栈对象的生命周期
    pt->print();
    (*pt).print();
    (pt.get())->print();
    pt.reset(new Point(1, 4));

    cout << endl;
    // RAII<Point> pt2 = pt;//error
    return 0;
}
```

#### `**auto_ptr**`

`auto_ptr`  可以执行复制，表面上执行了拷贝构造操作，实际上将被拷贝对象的资源转移了行对象，然后将被拷贝的对象置空，访问原被拷贝的对象会出错。

```cpp
#include <iostream>
#include <memory>

using std::cout;
using std::endl;
using std::auto_ptr;

void test(){
    //原生的裸指针
    int *pInt = new int(10);
    auto_ptr<int> ap(pInt);
    cout << "pInt = " << pInt <<endl;
    cout << "ap.get() = " << ap.get() << endl;
    cout << "*ap = " << *ap << endl;

    cout << endl;
    //表面上执行的是拷贝操作，但是底层已经发生了所有权的转移
    //并将ap的数据成员置位空了，该智能指针存在缺陷
    auto_ptr<int> ap2(ap);//可以进行复制
    cout << "*ap2 = " << *ap2 << endl;
    cout << "*ap = " << *ap << endl;
}

int main(int argc, char **argv){
    test();
    return 0;
}
```

#### `**shared_ptr**`

使用引用计数指向同一片内存空间

- 可以复制和赋值
- 同时具有移动语义和值语义
- 可以作为容器元素

循环引用 会导致内存泄漏，可以将其中一个`weak_ptr`和`shared_ptr` 共用解决，`weak_ptr`不会增加引用计数

#### `**weak_ptr**`

不会增加引用计数，属于弱引用

不能直接托管裸指针，只能初始化为为空 或 从`shared_ptr`进行复制或者赋值 或从其他的`weak_ptr` 复制或者赋值

使用`shared_ptr.use_count()`   `weak_ptr.expired()`

```cpp
#include <iostream>
#include <memory>

using std::cout;
using std::endl;
using std::shared_ptr;
using std::weak_ptr;

class Point
{
public:
    Point(int ix = 0, int iy = 0)
    : _ix(ix)
    , _iy(iy)
    {
        cout << "Point(int = 0, int = 0)" << endl;
    }

    void print() const
    {
        cout << "(" << _ix
             << ", " << _iy
             << ")";
    }

    ~Point()
    {
        cout << "~Point()" << endl;
    }

private:
    int _ix;
    int _iy;
};

void test()
{
    weak_ptr<Point> wp;

    {
        shared_ptr<Point> sp(new Point(1, 3));
        wp = sp;
        cout << "sp.use_count() = " << sp.use_count() << endl;
        cout << "wp.use_count() = " << wp.use_count() << endl;

        cout << endl;
        bool flag = wp.expired();
        if(flag)//use_count == 0
        {
            cout << "wp托管的资源不存在" << endl;
        }
        else
        {
            cout << "wp托管的资源存在" << endl;
        }

        shared_ptr<Point> sp2 = wp.lock();
        if(sp2 != nullptr)//true false   1  0 shared_ptr != nullptr
        {
            cout << "提升成功";
            sp2->print();
            cout << endl;
        }
        else
        {
            cout << "wp托管的资源不存在"  << endl;
        }
    }

    cout << endl << "离开作用域之后" << endl;
    shared_ptr<Point> sp2 = wp.lock();
    if(sp2)
    {
        cout << "提升成功";
        sp2->print();
        cout << endl;
    }
    else
    {
        cout << "wp托管的资源不存在"  << endl;
    }
}
int main(int argc, char **argv)
{
    test();
    return 0;
}
```

#### 使用智能指针存在的陷阱

使用不同的智能指针托管同一个裸指针( 堆空间 )

使用不同的智能指针托管同一块堆空间  ( `reset`  )

使用 两个`shared_ptr` 或者两个`unique_ptr`托管同一块空间

```cpp
#include <iostream>
#include <memory>

using std::cout;
using std::endl;
using std::unique_ptr;
using std::shared_ptr;

class Point
: public std::enable_shared_from_this<Point>
{
public:
    Point(int ix = 0, int iy = 0)
    : _ix(ix)
    , _iy(iy)
    {
        cout << "Point(int = 0, int = 0)" << endl;
    }

    void print() const
    {
        cout << "(" << _ix
             << ", " << _iy
             << ")";
    }

    /* Point *addPoint(Point *pt) */
    shared_ptr<Point> addPoint(Point *pt)
    {
        _ix += pt->_ix;
        _iy += pt->_iy;

        /* return shared_ptr<Point>(this); */
        return shared_from_this();
    }

    ~Point()
    {
        cout << "~Point()" << endl;
    }

private:
    int _ix;
    int _iy;
};

void test()
{
    //使用不同的智能指针托管同一个裸指针（堆空间）
    Point *pt = new Point(1, 2);
    unique_ptr<Point> up(pt);
    unique_ptr<Point> up2(pt);
}

void test2()
{
    unique_ptr<Point> up(new Point(1, 2));
    unique_ptr<Point> up2(new Point(3, 4));

    //使用不同的智能指针托管同一块堆空间
    up.reset(up2.get());
}

void test3()
{
    //使用不同的智能指针托管同一个裸指针（堆空间）
    Point *pt = new Point(1, 2);
    shared_ptr<Point> sp(pt);
    shared_ptr<Point> sp2(sp);//ok
    shared_ptr<Point> sp3(pt);//error
}

void test4()
{
    shared_ptr<Point> sp(new Point(1, 2));
    shared_ptr<Point> sp2(new Point(3, 4));

    //使用不同的智能指针托管同一块堆空间
    sp.reset(sp2.get());
}

void test5()
{
    shared_ptr<Point> sp(new Point(1, 2));
    cout << "*sp = ";
    sp->print();
    cout << endl;

    cout << endl;
    shared_ptr<Point> sp2(new Point(3, 4));
    cout << "*sp2 = ";
    sp2->print();
    cout << endl;

    cout << endl;
    shared_ptr<Point> sp3(sp->addPoint(sp2.get()));
    cout << "*sp3 = ";
    sp3->print();
    cout << endl;
}
int main(int argc, char **argv)
{
    test5();
    return 0;
}
```
