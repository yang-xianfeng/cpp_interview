### 4.2 scanf()  
int scanf(格式串，表达式1，表达式2，...)<br />`scanf( )` 本质上是一个“模式匹配函数”，试图把输入的字符与转换说明进行匹配，<br />转化成对应的二进制，写入到对应的内存空间中。

从左到右依次处理转换说明，如果成功，则继续处理后面的字符串；<br />如果失败，则立即返回，返回值表示处理转换说明 成功的个数。<br />**返回值 ：匹配成功的值的个数**

#转换说明：1. 表示匹配的规则；<br />2.表示将字符转化成对应的二进制数据。<br />格式串：普通字符：精确匹配<br />其他字符：精确匹配<br />空白字符：`**\space**``**\t**``**\n**``**\v**`**  ** 匹配任意多个空白字符，包括**零个**

注意：`scanf( )` 匹配到 %d , %f （进行**数值匹配**时），会**匹配**前面的空白字符。

#### 10.5.2 continue语句

- goto语句就没有这些限制，**唯一的限制**就是只能在同一个函数内进行跳转。

```c
int sum_arr(int matrix[][4], int n)

    int sum = 0;
    for (int i = 0; i < n; i++)
```

```c
long long fib1(int n) {
```

动态规划(DP):
```c
long long fib2(int n) {
	if (n == 0) return 0;
	if (n == 1) return 1;
	int a = 0, b = 1;
	for (int i = 2; i <= n; i++) {
		int tmp = a + b; // 第i项的值
		// 为计算第i+1项做准备
		a = b;
		b = tmp;
	}
	return b;
}
```

### 14.2 汉诺塔 (hanoi)
```c
void hanoi(int n, char start, char middle, char target) {
	// base case
	if (n == 1) {
		printf("%c --> %c\n", start, target);
		return;
	}
	// 递推公式:
    
    //把(n-1)个盘子从start移动到middle上
	hanoi(n - 1, start, target, middle);
	// 把最大的盘子从start移动到target上
	printf("%c --> %c\n", start, target);
    //把(n-1)个盘子从middle移动到target上
	hanoi(n - 1, middle, start, target);
}
```
移动次数：<br />S(n) =1 		       n=1<br /> = 2S(n-1) +1  n≥2<br />=> S(n) = 2n - 1

可以对任意类型的数组进行排列，不管元素类型是什么。

```c

typedef struct student_s {
	int number;
	char name[25];
	int chinese;
	int math;
	int english;
} Student;

int compare(const void* p1, const void* p2);

	Student students[5] = { {1, "liuyifei", 100, 100, 100}, {2, "wangyuyan", 99, 100, 100}, 
		{3, "zhaolinger", 100, 99, 100}, {4, "xiaolongnv", 100, 100, 99}, {5, "baixiuzhu", 100, 100, 99}};
	qsort(students, SIZE(students), sizeof(Student), compare);

}

// 比较规则: 总分从高到低, 语文成绩(高-->低), 数学成绩(高-->低), 英语成绩(高-->低), 姓名(字典顺序从小到大进行比较)
int compare(const void* p1, const void* p2) {
	Student* s1 = (Student*)p1;
	Student* s2 = (Student*)p2;
	int total1 = s1->chinese + s1->english + s1->math;
	int total2 = s2->chinese + s2->english + s2->math;
	if (total1 != total2) {
		return total2 - total1;//从大到小
        // total1 > total2 
        // total1 < total2
	}
	if (s1->chinese != s2->chinese) {
		return s2->chinese - s1->chinese;//从大到小
	}
	if (s1->math != s2->math) {
		return s2->math - s1->math;//从大到小
	}
	if (s1->english != s2->english) {
		return s2->english - s1->english;//从大到小
	}
	return strcmp(s1->name, s2->name);
}
```
